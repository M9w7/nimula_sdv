# Nimula – Login Screen
Login-Screen für die Webplattform Nimula, einem Startup mit der Mission, durch kurze Meditations- und Yoga-Einheiten den Büroalltag zu erleichtern.
Dieses Projekt wurde im Rahmen einer Bewerbungsaufgabe umgesetzt.

## Projektziel
Ziel war es, einen modernen, ruhigen und markenkonformen Login-Screen zu gestalten und mit HTML & CSS umzusetzen.

Der Screen soll:
- Die Markenwerte (Ruhe, Klarheit, Fokus, Leichtigkeit) widerspiegeln
- Vertrauenswürdig und hochwertig wirken
- Eine klare visuelle Hierarchie besitzen
- Responsiv für vorgegeben Viewpors funktionieren

## Design-Konzept
- Natürliche Farbwelt (Grün-Töne mild für Ruhe & Balance)
- Minimalistisches UI für Fokus
- Modernes, reduziertes Layout
- Responsive Design für bestimmte Bildschirmgrößen

## Verwendete Technologien
- HTML5
- CSS3
- JS

## Features
- Strukturierte und semantische HTML-Struktur
- Wiederverwendbare CSS-Variablen (Farben, Abstände)
- Responsive Layout
- Modernes, simples Button-Design mit Hover-Effekten

## Installation & Nutzung
### Repository klonen:
git clone [repo-url]

### Projekt öffnen
index.html im Browser öffnen (Shortcut: STRG + 1). Es sind keine zusätzlichen Abhängigkeiten notwendig!
